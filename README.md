# Game Engine

A custom engine for creating 2D and 3D games.

### Projects
Engine is in use in the following:
- Spider Solitaire - https://github.com/wjackson2112/Spider

### External Libraries
- [glfw](https://github.com/glfw/glfw) - OpenGL 3D API
- [glad](https://github.com/Dav1dde/glad) - GL Loader-Generator
- [glm](https://github.com/g-truc/glm) - OpenGL Math library
- [stb_image](https://github.com/nothings/stb/blob/master/stb_image.h) - Image loader
- [freetype](https://github.com/freetype/freetype) - Text API
- [assimp](https://github.com/assimp/assimp) - Open Asset Import Library
- [openal-soft](https://github.com/kcat/openal-soft) - OpenAL 3D audio API
